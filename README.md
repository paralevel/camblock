# CamBlock
Shortcut for iOS and macOS that installs a configuration [profile](https://developer.apple.com/documentation/devicemanagement/restrictions) that completely blocks activation of the built-in camera.

[Release 20260512](https://www.icloud.com/shortcuts/5fc1dc4c7a0542358f9154794d5a2bc2)

<sub>Uninstall:
<br>
On macOS, go to System Settings › General › Device Management, click the “CamBlock” profile, then press the minus button.
<br>
On iOS, go to Settings › General › VPN & Device Management, tap the “CamBlock” profile, then tap “Remove Profile”.</sub>
